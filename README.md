# Dogs2022
An MVVM dogs app using AndroidX/JetPack. Allows send SMS/MMS using modern API's. Requires setting the app as a default SMS app to send image MMS.

---- Tech used ----
- MVVM architecture
- Livedata
- Fragments, Activities, Navigation Manager
- Modern SMS & MMS (requires app to be default SMS app to send MMS)
- Modern Activity Result provider interface
